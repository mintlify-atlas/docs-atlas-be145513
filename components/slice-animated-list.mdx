## Overview

The `SliceAnimatedList` is the core carousel component that provides an animated horizontal slider with smooth transitions, scaling effects, and navigation controls. Items are arranged horizontally with the selected item centered and at full scale, while surrounding items are progressively scaled down based on their distance from the center.

### _initializeButtonAnimation()

```dart
void _initializeButtonAnimation() {
  _buttonAnimationController = AnimationController(
    vsync: this,
    duration: _buttonPulseDuration,
  )..repeat(reverse: true);

  _buttonAnimation = Tween<double>(begin: 1.0, end: 1.3).animate(
    CurvedAnimation(
      parent: _buttonAnimationController,
      curve: Curves.easeInOut,
    ),
  );
}
```

Creates a repeating pulse animation for navigation buttons.

## Scale Calculation

### _calculateItemScale(int index)

```dart
double _calculateItemScale(int index) {
  final int distanceFromCenter = (index - _selectedIndex).abs();
  return (_maxScale - (distanceFromCenter * _scaleFactor)).clamp(
    _minScale,
    _maxScale,
  );
}
```

Calculates the scale for an item based on its distance from the selected item:
- Distance 0 (selected): scale = 1.0
- Distance 1: scale = 0.8
- Distance 2: scale = 0.6
- Distance 3+: scale = 0.3 (clamped to minimum)

                left: 20,

                  icon: Icons.arrow_back_ios,
                  onPressed: _selectPreviousItem,
                  isLeft: true,
                ),
              ),

            // Right navigation button - only show if not at last item
            if (_selectedIndex < _items.length - 1)
              Positioned(
                right: 20,
                child: _buildNavigationButton(
                  icon: Icons.arrow_forward_ios,
                  onPressed: _selectNextItem,
                  isLeft: false,
                ),
              ),
          ],
        ),
      ),
    ],
  );
}

## Animation Details

<Tabs>
  <Tab title="Position Animation">
    Items use `AnimatedPositioned` with:
    - **Duration**: 300ms
    - **Curve**: `Curves.easeInOut`
    - **Offset**: Calculated as `(index - _selectedIndex) * _itemSpacing`
    
    The offset centers the selected item and positions others relative to it.
  </Tab>
  
  <Tab title="Scale Animation">
    Items use `AnimatedScale` with:
    - **Duration**: 300ms
    - **Curve**: `Curves.easeInOut`
    - **Scale**: Calculated by `_calculateItemScale(index)`
    
    Scale decreases by 0.2 for each item away from center.
  </Tab>
  
  <Tab title="Button Pulse">
    Navigation buttons use `AnimatedBuilder` with:
    - **Duration**: 1500ms
    - **Curve**: `Curves.easeInOut`
    - **Scale Range**: 1.0 to 1.3
    - **Mode**: Repeating with reverse
    
    Creates a subtle breathing effect to draw attention.
  </Tab>
</Tabs>
