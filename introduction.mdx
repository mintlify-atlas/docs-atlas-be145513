<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your animated carousel up and running in minutes
  </p>

  <Steps>
    <Step title="Add the widget to your app">
      Copy the `SliceAnimatedList` widget into your Flutter project's `lib` directory.

      ```dart
      import 'package:flutter/material.dart';
      import 'package:your_app/slice_animated_list.dart';

      class MyHomePage extends StatelessWidget {
        @override
        Widget build(BuildContext context) {
          return Scaffold(
            body: SliceAnimatedList(),
          );
        }
      }
      ```
    </Step>

    <Step title="Configure landscape orientation">
      Set your app to landscape mode in `main.dart` for the best carousel experience.

      ```dart
      void main() {
        WidgetsFlutterBinding.ensureInitialized();
        SystemChrome.setPreferredOrientations([
          DeviceOrientation.landscapeLeft,
          DeviceOrientation.landscapeRight,
        ]).then((_) {
          runApp(const MyApp());
        });
      }
      ```
    </Step>

    <Step title="Customize your items">
      Modify the `_buildCardItem` method to display your own content, images, or profile cards.

      ```dart
      Widget _buildCardItem(BuildContext context, int item, TextStyle? textStyle, double scale) {
        return SizedBox(
          width: _itemWidth,
          child: Card(
            color: Colors.primaries[item % Colors.primaries.length],
            child: Center(child: Text('Item $item', style: textStyle)),
          ),
        );
      }
      ```

      <Tip>Replace the `Text` widget with `CircleAvatar` for profile pictures or `Image.network` for product photos.</Tip>
    </Step>

    <Step title="Run your app">
      Launch your Flutter app and interact with the carousel by tapping items or using the navigation buttons.

      ```bash
      flutter run
      ```
    </Step>
  </Steps>
</div>
